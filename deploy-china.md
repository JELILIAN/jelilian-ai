- 拖拽整个项目文件夹

#### 3. 配置设置
- 项目名称：`jelilian-ai`
- 框架：`Other`
- 构建命令：`npm install`
- 输出目录：`.`

#### 4. 环境变量
```
NODE_ENV=production
```

#### 5. 部署完成
